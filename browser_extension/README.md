# Kevin's Second Brain － 瀏覽器擴充功能

Chrome Manifest V3 擴充功能，可將**目前網頁**或**快速筆記**一鍵存入第二大腦。

## 功能

- **存入此頁**：將當前分頁的網址送給後端，由後端抓取內容並經 AI 摘要後存入 Obsidian。
- **X (Twitter) 貼文**：若目前網址為 x.com 或 twitter.com，擴充功能會直接從頁面擷取貼文文字並存入（避免後端抓取到「JavaScript is not available」）。
- **快速筆記**：在彈出視窗輸入或貼上文字，一鍵以「純文字」存入第二大腦。
- **設定**：可設定後端 API 網址（本機 `http://localhost:8000` 或遠端 Win11 的 IP）。

## 安裝方式（開發者模式）

1. 確認後端已啟動（例如在 Win11 上執行 `docker-compose up`，或本機 `uvicorn`）。
2. 開啟 Chrome，網址列輸入 `chrome://extensions/`。
3. 右上角開啟 **「開發人員模式」**。
4. 點 **「載入未封裝項目」**。
5. 選擇本專案內的 **`browser_extension`** 資料夾（即包含 `manifest.json` 的目錄）。
6. 擴充功能會出現在工具列；可點圖釘固定。

## 使用方式

1. 瀏覽任意網頁後，點工具列上的 **Second Brain 擴充功能圖示**。
2. 若要存**目前網頁**：點 **「存入此頁」**。
3. 若要存**一段文字**：在「快速筆記」框輸入或貼上後，點 **「存入筆記」**。
4. 若後端在另一台電腦（例如 Win11）：點 **「設定」**，將 API 網址改為 `http://該電腦IP:8000`，再儲存。

## 注意事項

- 無法儲存 Chrome 內建頁面（如 `chrome://`、`edge://`）。
- 後端需已啟動且 CORS 允許擴充功能連線（本專案後端已設定允許）。
- 從 Mac 連到 Win11 後端時，請在擴充功能設定中填 Win11 的 IP（例如 `http://192.168.1.100:8000`）。
